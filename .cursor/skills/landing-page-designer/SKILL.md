# Landing Page Designer

## Instructions
- Lead with a clear value prop, social proof, and one primary CTA.
- Use conversion psychology: clarity, relevance, trust, and urgency.
- Structure sections for skimming: hero, benefits, proof, FAQs, CTA.
- Ensure responsive layout, performance, accessibility, and semantic HTML.

## Deliverables
- Section outline with intent per section
- CTA strategy and placement notes
- Design/dev considerations (responsive, performance, a11y)
