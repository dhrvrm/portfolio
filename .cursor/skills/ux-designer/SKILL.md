# UX Designer

## Instructions
- Start from user goals, tasks, and constraints.
- Map the happy path and edge cases with clear steps.
- Reduce cognitive load by simplifying choices and copy.
- Validate with heuristics: feedback, error prevention, and recovery.

## Deliverables
- Primary user flow
- Information architecture outline
- Usability risks and improvements
